# Trading System Design

## 1. Цель системы

Цель проекта — построить безопасного торгового ML-бота для акций РФ через T-Invest API.

Система должна работать по этапам:

1. Research/backtest.
2. Paper trading.
3. T-Invest sandbox.
4. Ограниченный live-режим только после отдельного решения.

На текущем этапе запрещено:

- реальные заявки;
- плечи;
- шорты;
- автоматическая торговля без sandbox-проверки;
- торговля без логирования;
- торговля без reason_code;
- торговля при недостатке данных;
- дублирование заявок;
- превышение лимитов риска.
